# 📇 Contact Management System
 **Live Project:** [Link](https://contact-management-system-xo5v.onrender.com)

A simple and user-friendly Contact Management Application built using Flask.  
This application allows users to perform full CRUD operations with proper validation.

---

## 🚀 Features

- ➕ Add new contacts  
- 📋 View all contacts  
- ✏️ Update existing contacts  
- ❌ Delete contacts  
- ✅ Email format validation  
- 📱 Phone number validation (10 digits)  
- 🚫 Prevent duplicate email entries  
- 🎨 Clean UI using Bootstrap  

---

## 🛠️ Tech Stack

### Backend
- Python (Flask)

### Frontend
- HTML, CSS, Bootstrap

### Database
- SQLite (for local development)
- PostgreSQL (for deployment on Render)

### ORM
- Flask-SQLAlchemy

### Deployment
- Render

### Containerization (Bonus)
- Docker
